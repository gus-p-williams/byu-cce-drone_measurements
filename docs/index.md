# BYU CCE - Drone Measurements

Welcome to the content repository for **BYU CCE - Drone Measurements**. This repository contains the homework assignments and exercises associated with the class. The content is organized by unit and category and can be browsed using the navigation bar on the left. You can also access this content from the Schedule page on the BYU Learning Suite page for the class.  The schedule will contain links to the content for each unit and category. This content is maintained by the course instructors and TAs and is subject to change. If you have any questions, comments, or suggestions, please contact the course instructors.
